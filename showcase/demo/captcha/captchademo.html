Captcha Captcha is a form validation component based on Recaptcha API V2.

Import


import {CaptchaModule} from 'primeng/primeng';

Getting Started

Captcha is initialized with a siteKey and an event for producing a response.


<p-captcha siteKey="YOUR_SITE_KEY" (onResponse)="showResponse($event)"></p-captcha>

Attributes

Name Type Default Description
sitekey string null Your sitekey.
theme string light Optional. The color theme of the widget.
type string image Optional. The type of CAPTCHA to serve.
size string normal Optional. The size of the widget.
tabindex number 0 Optional. The tabindex of the widget and challenge. If other elements in your page use tabindex, it should be set to make user navigation easier.
language string en Language of the widget.

Events

Name Parameters Description
onResponse event.originalEvent: Browser event Callback to invoke when captcha succeed.
onExpire event: Browser event Callback to invoke when captcha expires.

Methods

Name Parameters Description
render container,parameters Renders the container as a reCAPTCHA widget and returns the ID of the newly created widget. container The HTML element to render the reCAPTCHA widget. Specify either the ID of the container (string) or the DOM element itself. parameters An object containing parameters as key=value pairs, for example, {"sitekey": "your_site_key", "theme": "light"}.
reset opt_widget_id Resets the reCAPTCHA widget. opt_widget_id Optional widget ID, defaults to the first widget created if unspecified.
getResponse opt_widget_id Gets the response for the reCAPTCHA widget. opt_widget_id Optional widget ID, defaults to the first widget created if unspecified.
